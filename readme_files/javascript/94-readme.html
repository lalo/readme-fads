Fine Uploader aims to make file-uploading on the web possible in every browser and mobile device. It is cross-browser, dependency-free, and 100% Javascript.

FineUploader is simple to use. You only need to include one JavaScript file. There are absolutely no other dependencies. For more information, please see the documentation.
