Scala IDE for Eclipse

Scala IDE provides advanced editing and debugging support for the development of pure Scala and mixed Scala-Java applications inside Eclipse

Scala IDE is Open Source and available under the Scala License.

Learn more at http://scala-ide.org.

Report A Bug

File a ticket in our issue tracker, or drop a message in the scala-ide-user mailing list if you have any doubt.
