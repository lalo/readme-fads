IMPORTANT NOTE!!!

Curator has moved to Apache. The Netflix Curator project will remain to hold Netflix extensions to Curator.

curator.incubator.apache.org

The previous Netflix branch is now in a branch named “archive”.

DESCRIPTION

Netflix Curator contains extensions to Apache Curator.

DETAILS

Please see the doc at github.com/Netflix/curator/wiki

BUILDING

Curator is built via Gradle (www.gradle.org). To build from the command line:

./gradlew build

ARTIFACTS

Netflix Curator binaries are published to Maven Central. Please see the docs for details.
