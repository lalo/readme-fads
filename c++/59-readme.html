GamePlay v2.0.0

GamePlay3D is an open-source, cross-platform 3D native C++ game framework making it easy to learn and write mobile and desktop games.

  • Forums

Supported Platforms

  • Microsoft Windows (using Microsoft Visual Studio)
  • Apple MacOS X (using Apple XCode)
  • Linux (using CMake)
  • Apple iOS (using Apple XCode)
  • Google Android (using Google Android NDK)
  • BlackBerry (using BlackBerry Native SDK)
