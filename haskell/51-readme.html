Welcome to pronk

pronk consists of a library and a command line application for load testing web servers. You can think of the command line tool as similar to httperf or ab (also known as apachebench), only more modern and simpler to deal with.

Join in!

We are happy to receive bug reports, fixes, documentation enhancements, and other improvements.

Please report bugs via the github issue tracker.

Master git repository:

  • git clone git://github.com/bos/pronk.git

There's also a Mercurial mirror:

  • hg clone http://bitbucket.org/bos/pronk

(You can create and contribute changes using either git or Mercurial.)
